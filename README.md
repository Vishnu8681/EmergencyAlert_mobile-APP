# 🚨 Emergency Alert App

A mobile app that sends a distress signal and live GPS location to pre-saved contacts with a single tap.

## Features
- ✅ One tap SOS button
- ✅ Live GPS location tracking
- ✅ Send SMS with location link
- ✅ Save emergency contacts
- ✅ Built with React Native & Expo

## Tech Stack
- React Native
- Expo
- GPS Location
- SMS Integration

## How to Run
cd EmergencyAlert4

npm start
